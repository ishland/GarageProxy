## GarageProxy
Forwards anything from LAN.

### Requirements
- PHP 5.3 or higher
- A POSIX compatible operating system (Linux, OSX, BSD)
- POSIX and PCNTL extensions for PHP

- Run `php GarageProxy.phar start [-d]`
- Enjoy.
